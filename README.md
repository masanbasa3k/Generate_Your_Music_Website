# AI-Generated MIDI File Generator

This project includes an artificial intelligence model that generates MIDI files. It also contains a Flask application that makes this AI model available on a website.

## Getting Started

To run this project in your local development environment, follow these steps:

1. Create and activate a virtual environment in the project's root directory (optional).

   ```bash
   python -m venv venv
   source venv/bin/activate
   ```

2. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Start the Flask application:

   ```bash
   python main.py
   ```

4. Visit `http://127.0.0.1:5000` in your browser to access the project.
